# ⚓ ShipStream-AI: Hull Lines Design Module
**The Foundation of a Future-Ready Automated Naval Architecture Suite.**

## 🚢 Overview
This module represents the first phase of an ambitious, web-based integrated platform designed to automate the core workflows of Naval Architecture and Marine Engineering. Currently, this phase focuses on the **Complete and Advanced Hull Lines Design**, providing precision geometric modeling that serves as the baseline for all subsequent naval calculations.

Our vision is to evolve this tool into a comprehensive Web App that handles everything a naval architect needs in a single environment.

## 🛠 Project Roadmap
This project is being developed in stages to cover the full lifecycle of ship design:
*  **Phase 1: Intelligent Hull Generation** (Current) - Advanced mathematical modeling of hull surfaces. (done)
*  **Phase 2: Hydrostatics & Stability** - Automated GZ curve calculation and large angle stability analysis. (pending)
*  **Phase 3: Resistance & Powering** - Resistance prediction using standard series and regression. (pending)
*  **Phase 4: Structural & Marine Systems** - Weight estimation and piping layout automation. (pending)

## Technical Features
* **Geometric Precision:** Curvature-continuous modeling for fair hull lines.
* **Modular Logic:** Separate engines for `geometry`, `hydro`, and `surface` calculations for easy scaling.
* **Interactive Visualization:** High-performance rendering of Body Plans, Sheer Plans, and Half-Breadth Plans directly in the browser.
* **Web-Native:** Zero installation required; runs entirely on modern web standards (JS/HTML5/CSS3).
